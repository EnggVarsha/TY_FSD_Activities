# 📄 Resume Builder Application

A modern, interactive web application that helps users create professional resumes with ease. Built with HTML, CSS, and JavaScript, this tool features a live preview, multiple templates, AI-powered suggestions, and a resume scoring system.

![Resume Builder Preview](https://via.placeholder.com/800x400?text=Resume+Builder+Preview)

## ✨ Features

### 🖼️ Profile Picture Upload
- Upload and preview profile photos
- Photo appears in resume preview
- Automatic image cropping and circular display
- Remove photo option
- Photo persistence in browser storage

### 📝 Multi-Step Form
- **Step 1:** Personal Information (with photo upload)
- **Step 2:** Work Experience
- **Step 3:** Education
- **Step 4:** Skills
- **Step 5:** Professional Summary

### 🎨 Multiple Templates
- **Classic** - Traditional Times New Roman style
- **Modern** - Clean Helvetica with colored sidebar
- **Minimal** - Simple Arial with borders
- **Professional** - Elegant Georgia with top/bottom accents

### 🤖 AI-Powered Suggestions
- **Smart Skills Generator** - One-click skill suggestions
- **Experience Enhancer** - Professional bullet points
- **Auto Summary Generator** - Creates tailored summaries based on your input

### 📊 Resume Score Calculator
- Comprehensive scoring system (0-100 points)
- Real-time feedback and suggestions
- Color-coded results (Green/Yellow/Red)
- Detailed improvement checklist

### 💾 Auto-Save & Persistence
- Automatically saves data to localStorage
- Resume data persists after browser refresh
- Photo data stored locally

### 📥 PDF Export
- One-click PDF download
- High-quality rendering
- Preserves all formatting and photos
- Letter-size format
